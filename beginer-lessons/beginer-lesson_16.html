Simple Sentences: "This is a book."

Simple Sentences Nima?

Ingliz tilida "simple sentences" oddiy gaplar bo‘lib, bitta subyekt (kim yoki nima haqida gapirilayotgani) va predikat (harakat yoki holatni bildiruvchi so‘z)dan iborat bo‘ladi. Bu gaplar qisqa va sodda bo‘lib, yangi boshlovchilar uchun mosdir.

Oddiy Gaplarning Tuzilishi

Oddiy gaplar quyidagi tuzilishga ega:

  • Subject + Verb + Object: She reads a book.
  • Subject + Verb: He runs.
  • Subject + Verb + Adjective: The car is red.

Oddiy Gaplar Misollari

Quyida oddiy gaplarning misollari keltirilgan:

  • This is a book. (Bu kitob.)
  • She is a teacher. (U o‘qituvchi.)
  • The dog is black. (It qora.)
  • I like apples. (Men olmani yoqtiraman.)

"This", "That", "These", "Those"

Oddiy gaplarda ko‘rsatish olmoshlari ("demonstrative pronouns") ishlatilishi mumkin:

  • This: yaqin joydagi bitta predmet (e.g., This is a pen.)
  • That: uzoq joydagi bitta predmet (e.g., That is a car.)
  • These: yaqin joydagi ko‘plikdagi predmetlar (e.g., These are books.)
  • Those: uzoq joydagi ko‘plikdagi predmetlar (e.g., Those are apples.)

Oddiy Gaplarni Ishlatish

Oddiy gaplar kundalik muloqotda keng qo‘llaniladi. Misollar:

  • Javob berishda: This is my bag.
  • Tanishtirishda: She is Anna.
  • Tavsif berishda: The weather is nice today.

Tavsiyalar

Oddiy gaplarni o‘zlashtirish uchun:

  • Ko‘rsatish olmoshlarini mashq qiling ("this", "that", "these", "those").
  • O‘z hayotingizdagi oddiy predmetlarni tasvirlash uchun gaplar tuzing.
  • Gaplarning tuzilishini mustahkamlash uchun yozma va og‘zaki mashqlar bajaring.

Mashqlar: Simple Sentences: "This is a book."

1. Gaplarni To‘ldiring

Quyidagi gaplarni mos so‘zlar bilan to‘ldiring:

  • This is _______. (e.g., a pen)
  • That is _______. (e.g., a dog)
  • These are _______. (e.g., apples)

2. Ko‘rsatish Olmoshlarini Tanlang

Quyidagi gaplar uchun mos ko‘rsatish olmoshini tanlang:

  • _______ is a car. (This / That)
  • _______ are chairs. (These / Those)
  • _______ is my house. (This / That)

3. Gaplarni Yozing

Quyidagi predmetlar bilan oddiy gaplar yozing:

  • table
  • cat
  • books

4. To‘g‘ri Gaplarni Tanlang

Quyidagi gaplardan to‘g‘risini belgilang:

  • This are a book. / This is a book.
  • That is pens. / Those are pens.
  • These is my friends. / These are my friends.

5. Gaplarni Tuzing

Quyidagi so‘zlarni ishlatib oddiy gaplar tuzing:

  • red, car
  • happy, child
  • new, phone

6. Gaplarni Tahrirlang

Quyidagi gaplarda xatolarni topib, to‘g‘rilang:

  • This is books.
  • These is my teacher.
  • That are a dog.

7. Gaplarni To‘g‘ri Tartibda Joylashtiring

Quyidagi gaplarni so‘zlarni to‘g‘ri tartibda joylashtiring:

  • is / this / apple / an
  • car / is / that / a
  • are / those / chairs

8. Ko‘plik va Yakkalikni Farqlang

Quyidagi gaplarni ko‘plik yoki yakkalik shaklida qayta yozing:

  • This is a book. → _______
  • These are pens. → _______
  • That is a chair. → _______

9. Gapni To‘ldiring

Quyidagi gaplarni mos so‘zlar bilan to‘ldiring:

  • This is a _______.
  • Those are _______.
  • That is a _______.

10. Dialog Tuzing

Quyidagi iboralar asosida qisqa dialog yozing:

What is this? / This is a phone. / What about that?
